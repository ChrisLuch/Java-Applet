Instantaneous Velocity

The velocity of an object at an instant of time is defined mathematically as the limit of an average velocity. The definition is analogous to that of instantaneous speed. See Speed/Instantaneous/Explain It.

First one calculates the average velocity over a time interval Dt containing the given instant. Then one shrinks Dt down to zero size so that it reduces to the given instant.
