The Wheatstone Bridge

Let us choose positive current directions as indicated on the left. The four junctions in the circuit have been labeled A to D. The three clockwise loops for which we will write down Kirchhoff's loop equations are indicated as well. We will write down Kirchhoff's junction equations for junctions A to C. The equation for junction D will give no new information.

Junction A. I0 = I1 + I2

Junction B. I1 = I3 + I5

Junction C. I0 = I3 + I4

If the terminal potential difference of the battery is denoted V0, the three loop equations are

Loop 1. V0 - R1I1 - R3I3 = 0

Loop 2. -R2I2 + R5I5 + R1I1 = 0

Loop 3. -R4I4 + R3I3 - R5I5 = 0.

It is quite a challenge to solve these equations. Your course will probably not require you to do something as lengthy as this. You may want to try it anyway. However, the main thing for you to learn here is to set up the equations. The solution for current I5 will be given on Page 10.

Note that the simulation does exactly what is outlined here. First it sets up Kirchhoff's current and voltage equations and then it solves them. It does this for circuits of arbitrary complexity.
