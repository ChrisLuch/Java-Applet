(a) Circular Motion

To start the motion, click anywhere in the simulation window. To reset, click a second time. The simulation shows positions of the object equal time intervals apart.

Our object (red) is performing circular motion while interacting with a second object (yellow).

This kind of motion can be observed when a planet (red) is revolving around the sun (yellow). The simulation is not to scale.

The red object's acceleration is not zero because the direction of the object's velocity is changing all the time. If the velocity is changing, there is acceleration.

Note that the object's speed is constant in this case. However, the acceleration is still unequal to zero because the velocity is changing.
