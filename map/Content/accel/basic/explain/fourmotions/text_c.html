(c) Linearly Accelerated Motion

To start the motion, click anywhere in the simulation window. To reset, click a second time. The simulation shows positions of the object equal time intervals apart.

The direction of the velocity is constant in this case, but the magnitude of the velocity is not, because the distances between the positions of the object as it moves down the incline are increasing.

Thus, this motion has non-zero acceleration because the velocity is not constant.
