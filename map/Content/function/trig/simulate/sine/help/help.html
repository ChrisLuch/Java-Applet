Explanation of Features

  1. Unit Circle
    Click and drag anywhere within the unit circle to set theta.
  2. Play
    Press this button to increment theta automatically.
  3. Reset
    Pressing this button will reset theta to zero.
  4. Show Graph
    Toggle this button to show and hide the sine wave graph.
  5. Help
         Pressing this button will show this help window.
