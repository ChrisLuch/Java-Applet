Multiplication of a Vector by a Scalar

If a vector is given in terms of its scalar components,

= (ax,ay),

then multiplying the vector by the scalar c is accomplished by multiplying each of its components by c. Thus,

c = (cax,cay).

E.g., if = (-4,3) and c = -2, then

-2 = (-2) × = (-2) × (-4, 3) = ((-2)×(-4), (-2)×3) = (8, -6).

This example uses a vector in two dimensions. Scalar multiplication works analogously in any other dimension.
