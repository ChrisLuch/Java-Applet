Multiplication of a Vector by a Scalar

The scalar multiplication of vectors obeys the general rule:

a(b) = (ab) = b(a),

where is a vector and a and b are two scalars. E.g.,

3(-5) = (-15) = -5(3).

You can verify this graphically by drawing the various vectors.

When a vector is multiplied by the number 0, one obtains a vector of magnitude zero. This vector, represented graphically by a point, is called the null vector. Its symbol, 0, looks just like the symbol for the scalar zero. One does not use an arrow above '0' to indicate that one is dealing with the null vector rather than the number zero, because it will be clear from the context which is which.

The vector - is defined so that + (-) = 0, where 0 is the null vector.

There is only one null vector. All vectors, no matter what their direction, will give the same null vector after multiplication by the number 0. A geometrical point has no direction.

Multiplying a vector by the number 1 gives again the same vector .
