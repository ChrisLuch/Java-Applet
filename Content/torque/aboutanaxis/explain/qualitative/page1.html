Torque About an Axis

Preparation: Torque/Basic Facts/Explain It.

The quantity that makes an object rotate about an axis is called the torque about that axis. When an object rotates about an axis of fixed direction (the axis itself does not need to be fixed), then it is sufficient to work with the scalar component of the torque in the direction of this axis, also called the torque about this axis.

For a general rotation in three dimensions that is not about an axis whose direction is fixed, the explanation of the motion requires torques that are vectors. When an object rotates about an axis of fixed direction (the axis itself does not need to be fixed), then it is sufficient to work with the scalar component of the torque in the direction of this axis, also called the torque about this axis.

Being the scalar component of a vector, the torque about an axis can take on both positive and negative values.

In this section, we will consider only rotations about an axis fixed in direction so that only the torque about this axis is needed, and not the torque vector. We will follow common practice in such cases, which is to refer to 'the torque about the axis' simply as the torque, for brevity.
