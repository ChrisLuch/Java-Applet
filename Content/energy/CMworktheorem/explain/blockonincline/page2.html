CM-Work Theorem Applied To A Block On An Incline

Answer. We can calculate the block's final speed v, which is equal to the speed of the block's CM, if we know the block's final translational kinetic energy KE, which can be calculated by using the CM-work theorem.

The amounts of CM-work WCM,a done by the applied force and WCM,fr done by the friction force are equal to

WCM,a = 5.81 J

and

WCM,fr = -1.62 J .

The calculations are done on Page 3. The CM-work done by the normal force is zero.

The CM-work WCM,g done by the gravitational force can be calculated from the block's vertical elevation change, which is equal to the y-component of the block's displacement, dy = d sin θ = 2 x sin 200 = 0.68 m.

WCM,g equals

WCM,g = Fg,y Δy = (-mg) Δy

= -0.4x10x0.68 = -2.74 J.

The CM-work theorem gives

ΔKE = WCM,a + WCM,fr + WCM,g

= 5.81 - 1.62 - 2.74

= 1.45 J .

Therefore,

KEfin = KEin + ΔKE

= (m/2)vin2 + ΔKE

= (0.4/2)x22 + 1.45

= 2.25 J.

From this we can obtain the final speed as

vfin = [2KEfin/m]1/2

= [2x2.25/0.4]1/2

= 3.35 m/s .

You can check this value by running the simulation with the given data.
