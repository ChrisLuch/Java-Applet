Help - Stopwatch
The applet simulates a stopwatch.

Click on a number to view the explanation of the corresponding item.

  1. Start/Stop

    2. To start the stopwatch, move the mouse pointer anywhere onto the green button field labeled "Start" and press the mouse button. The clock will start running and show the time elapsed in the format "hours:minutes:seconds". Seconds are shown to two decimals. The "hours" field is included in the data format only when at least one hour has elapsed.

    While the clock is running, the main clock button is displayed in red and labeled "Stop". To stop the clock, press the mouse button while the mouse pointer is in the red area. The main clock burn turns back into the green Start button. To resume, without having the time reset to 0, start the clock again. Etc.

  2. Reset

    3. Pressing the Reset button at any time, whether the clock is running or not, will cause the clock to be reset to 0.
