Physics Applets And Flash Animations

Calgary, Alberta, January 19, 2023

The Java applets and animations listed below were initially designed as part of a project titled Modular Approach to Physics by the Calgary Team, which was part of a collaboration of Alberta physicists. They can be used at the high school and first and second-year university levels for non-commercial purposes. The work is underway to update them for current internet browsers. The applets/animations which have been updated are indicated by the word "updated" in the parentheses beside their name. The animations have been converted from Flash to mp4.

Some applets may not be displayed automatically when you click on the link. The link may take you to a directory instead. In this case, click on the file "applet.html" (or, perhaps, "page2.html") in the directory to display the applet.

Most applets have a Help menu to provide information on how to use the applet. To access this information, pull down the Help menu and go to Applet Help. For some applets, Applet Help still has to be made available on the Help menu. The listing below indicates if an applet has Applet Help information available.

The information available from a Help menu under Applet Help generally consists of several files named Help, Assumptions, ShowMe, and Lesson.

  • Help provides information on how to use the applet.
  • Assumptions spells out the physical assumptions underlying the applet.
  • ShowMe is a guided tour through the applet in the form of exercises.
  • Lesson contains several exercises to be carried out with the applet that illustrates what can be learned with the applet.
